# Imágenes

## Logo principal del club

Coloca aquí el escudo del club con el nombre exacto:

```
public/img/logo-cva.png
```

Se muestra en la barra superior (header) de todas las páginas, a la izquierda
del texto "CVA · Quiniela · Mundial 2026".

Recomendaciones:
- Formato PNG con fondo transparente.
- Que se vea bien a una altura de ~40 px (p.ej. 160×160 px o mayor, cuadrado).
- Si usas otro nombre o formato (.svg, .jpg), actualiza la ruta en
  `views/partials/header.ejs` (atributo `src` de `img.brand-mark`).
